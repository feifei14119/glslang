1. 调试方法，
	1.1 在glslang工程的生成后事件中添加
		xcopy /y UserM_Debug\generic\glslang.* UserM_Debug，
		即将生成的dll文件从User_Debug\generic拷贝到User_debug目录中
	1.2 在StandAlong工程的调试选项卡中
		设置工作目录为F:\work\glslang\UserM_Debug\
		设置命令为StandAlone.exe
		设置参数为shader文件路径和文件名
2. 使用方法
	2.1 cd到StandAlone目录中
	2.2 ./StandAlone.exe
		-i : 打印语法树
		-m : dump link 结果（无效）
		-a : dump assembly 结果（无效）
3. 文件目录组织
	3.1 glslang/Include/ 包含glslang内部共享头文件
	3.2 glslang/Public/  包含对外的头文件
4. 使用范例
	4.1 IntermOut.cpp:  遍历语法树
	4.2 CodeGen.cpp:    后端的基本接口
	4.3 Link.cpp:       链接器的基本接口
	4.4 Intermediate.h: 语法树的数据结构
